# Assign Multiple Values

Pythonda bir nechta qiymatlarni bir nechta o'zgaruvchilarga bitta qatorda berish mumkin.

# Assigning the Same Value

```python
a = b = c = 100
print(a, b, c)
```

# Assigning Different Values

```python
a, b, c = 20, 2.5, "Python"
print(a)  # 20
print(b)  # 2.5
print(c)  # "Python"
```
